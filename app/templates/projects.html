Zeromon

  • Started: December 20 (Thursday), 2018
  • Langage: Ansible
  • I absolutely love Zabbix as a monitoring tool so a friend of mine (Kyle Parisi) came up with the idea of a project that we call Zeromon.
  • The goal was to create an Amazon Machine Image (AMI) to allow any one to deploy a fully working Zabbix installation on EC2 with a single click from the Amazon Web Services Marketplace which we did!
  • Zeromon uses a combination of cloud-init along with our open-sourced Ansible playbook and roles to make setting up Zabbix on Amazon as simple as possible.

gaw.sh URL Shortener

  • Started: December 2 (Friday), 2011
  • Deprecated: Dec 28 (Sunday), 2014
  • Language: PHP, MySQL, HTML, CSS, JavaScript
  • I originally purchased the domain name "gaw.sh" as a short address to remember quickly and to have some fun with.
  • However, I eventually realized that it would make a great domain name for a URL shortener so I began creating my own.
  • This was deprecated due to a number of reasons including unmaintainability. URL shorteners attract a large amount of abuse and the public repository no longer matches the numerous changes (such as recaptcha) that were added to avoid continued spam.
  • The website itself will stay up as long as possible, but further changes are not going to be made public since the repository is no longer in-sync.
